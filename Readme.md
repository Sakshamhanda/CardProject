# 📦 Card Project

A simple and responsive **Card UI design** built using **HTML & CSS**.
This project demonstrates how to create a clean layout with proper alignment, spacing, and styling using Flexbox.

---

## 🚀 Features

* 🎨 Beautiful gradient background
* 📱 Centered card layout using Flexbox
* 🖼️ Image with rounded styling
* 🔗 Clickable title (redirects to GitHub profile)
* 📝 Description text with proper alignment
* 💡 Clean and beginner-friendly code

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox)

---

## 📸 Preview

This card includes:

* A logo/image at the top
* A title with a clickable link
* A short description

---

## 🧠 What I Learned

* How to use **Flexbox** for layout alignment
* Managing spacing without using unnecessary padding
* Styling images using `border-radius`
* Creating responsive and clean UI components
